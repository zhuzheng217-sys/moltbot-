# Lunar Reminder Bot 

This repository contains a bot that generates lunar calendar reminders for specific events.
1. 如果是作为单独的日历导入的（如 Google 日历 Web 导入 .ics 文件的方式）
这种方式一般新建了一个和文件同名的日历。
删除方法：
打开 Google 日历网页版。
左边“其他日历”区域，找到你导入的那个日历（名字一般和你导入的文件名类似，比如 Lunar_Reminders_2024）。
鼠标悬停→点击右侧 “⋮” → 选择“设置”。
滚动到底，点击“删除日历”或“取消订阅”，确认即可。
手机上同步后会自动消失。
2. 如果是导入所有提醒到“主日历”或已有日历（如手机本地批量导入，直接 Event 全部加到主日历）
这种方式是所有提醒插入现有日历，事件实际和你其它日历混在一起，不是单独日历。
没有“一键删除”，但可以按以下方式批量清理：
A. 用 Google 日历网页版批量删除
搜索框输入 明天是农历（如果你的提醒都以这个开头，系统会筛选所有相关事件）。
批量点击或按住 Ctrl 多选，删除。
或者用“脚本”自动删除（见后文）。
B. 用手机日历 App 删除
在事件列表内搜索关键词，逐一全部删除。
C. 脚本删除（Google 日历支持 API 批量删除）
如果事件很多且无法手动清理，可借助 Google Calendar API 和 Python 脚本，用关键词（如“明天是农历”）批量清理。用户需配置授权(App Password/OAuth)等。
需要帮助写这个脚本可以告知。
3. 本地系统日历（如三星/小米/OPPO等本地自带日历App）
进入App，查找所有用“明天是农历”开头的事件，逐一全选后删除即可。
部分日历（如华为日历）也支持批量操作：长按事件列表，多选后批量删。
总结
如是“单独的日历”，直接整历删除最简单。
如是“主日历批量事件”，需按关键词手动搜索或批量删除。
如事件非常多且手动不便，可用 Google Calendar API 脚本自动批量删，需要的话我可以提供脚本代码。
请告知你的导入方式（单独日历/主日历/其他），我能帮你定向给出详细的删除方案或脚本。
